MirrorX

Remote control tool for enterprise, teams and individuals.

Security

Support end-to-end encryption natively keeps all data and behaviours under user control.

Fast

Support hardware-accelerated screen acquisition and rendering. Allow p2p direct connections within LAN, minimize interaction delay.

Open Source

MirrorX is a COMPLETELY open source project. Not only client-side of MirrorX, but also includes server-side programs.
